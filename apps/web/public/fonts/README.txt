General Sans, Variable, by Indian Type Foundry, distributed via Fontshare under
the Fontshare Free License.

Drop GeneralSans-Variable.woff2 into this directory before the first build.
The font is free to use; download from https://www.fontshare.com/fonts/general-sans.

Fallback: if the file is missing, fonts.ts falls back to Inter Variable so the
build still ships. The display fallback is documented in apps/web/app/fonts.ts.
